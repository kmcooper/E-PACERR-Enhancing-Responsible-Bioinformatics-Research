# E-PACERR Educational Modules
This is the code repository for the E-PACERR: ENHANCING PROFESSIONALISM, ADVOCACY, AND CAPACITY FOR EXCELLENCE IN RESPONSIBLE BIOINFORMATICS RESEARCH project.
The goal is to provide a free, open, self-paced curriculum supporting excellence, reproducibility, and professionalism in bioinformatics research.
