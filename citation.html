How to Cite this Work

If you use or adapt materials from this repository, please cite it as:

Cooper, K. M. (2026). *E-PACERR: Enhancing Responsible Bioinformatics Research* [GitHub repository]. GitHub. https://github.com/kmcooper/E-PACERR-Enhancing-Responsible-Bioinformatics-Research.

or you can use BibTex : 

        
@misc{cooper2026epacerr,
  author       = {Cooper, Kate M.},
  title        = {E-PACERR: Enhancing Responsible Bioinformatics Research},
  year         = {2026},
  howpublished = {\url{https://github.com/kmcooper/E-PACERR-Enhancing-Responsible-Bioinformatics-Research}},
  note         = {GitHub repository}
}
