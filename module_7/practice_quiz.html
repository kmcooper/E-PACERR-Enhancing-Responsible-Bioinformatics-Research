Practice Quiz: Databases, Information Management, and Big Data

Select the best answer for each question.

1. What is a database?


Correct: A database stores records systematically for retrieval.
2. What is an example of a flatfile database?


Correct: Flatfile databases include spreadsheets and text-based tables.
3. What distinguishes a relational database?


Correct: Relational databases use multiple linked tables.
4. What is a primary key?


Correct: A primary key uniquely identifies each record.
5. CRUD operations include:


Correct: CRUD stands for Create, Retrieve, Update, Delete.
6. What does a LIMS system do?


Correct: LIMS manages lab workflows, samples, and data.
7. When is a LIMS most useful?


Correct: LIMS is best for repetitive, large-scale work.
8. What is big data?


Correct: Big data refers to large, complex datasets.
9. What is a data warehouse used for?


Correct: Data warehouses support long-term analysis and decisions.
10. What is a data lake?


Correct: Data lakes store raw data until needed.
