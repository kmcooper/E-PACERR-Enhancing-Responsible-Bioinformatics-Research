How to use this checklist

This checklist is intentionally basic and is designed for students, beginner trainees, or anyone who is learning how to design a bioinformtics project that is organized, transparent, and easier for others to reproduce.

Ideally, you would use this in the design stage of your project.

Good place to start

  • Talk to your collaborators early.
  • Name files clearly.
  • Save your code in one place.
  • Write down what you changed and why.
  • Make sure someone else can follow your workflow.
